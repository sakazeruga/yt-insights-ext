知識のデータベースに

Gemini AIが動画を自動分析。要約・キーポイント・タイムスタンプ・知見を抽出して、あなただけのライブラリを構築します。

⬇ ダウンロード（ZIP） ⭐ GitHub

Features

見るだけで終わらせない

ブックマークした瞬間にGeminiが分析を開始。あとで見返すための情報を自動で整理します。

📝
AI自動要約
Gemini 3.5 Flashが動画全体を読み込み、200〜300字の日本語要約を生成します。
✅
キーポイント抽出
動画の重要なポイントを最大10個の箇条書きで整理。内容を素早く把握できます。
⏱
タイムスタンプ
重要なシーンのタイムスタンプを自動抽出。クリックするとその場面へジャンプ。
💡
実践的な知見
動画から得られる学び・次のアクションをAIが提案。知識を行動につなげます。
🏷
AIタグ＋手動タグ
AIが自動でタグを付け、手動タグも追加可能。カテゴリ分類で整理も簡単。
☁️
クラウド保存
さくらのレンタルサーバー（MySQL）に保存。複数端末からライブラリを参照できます。

Setup

5ステップで始める

Gemini APIキー（無料）とさくらのレンタルサーバーがあれば動作します。

1
ZIPをダウンロード・展開
上のボタンからZIPをダウンロードして、任意のフォルダに展開してください。
2
Gemini APIキーを取得
Google AI Studio で無料のAPIキーを取得してください。使用モデル: gemini-3.5-flash
3
さくらのMySQLをセットアップ
phpMyAdminで api/schema.sql を実行 → api/config.php にDB情報を記入 → FTPで api/ フォルダをアップロード。
4
Chrome拡張機能をインストール
chrome://extensions → デベロッパーモードON → 「パッケージ化されていない拡張機能を読み込む」でフォルダを選択。
5
オプションページで設定
拡張機能アイコンを右クリック → オプション → Gemini APIキー・API URL・API Keyを入力して保存。
